Dibuat untuk menyelesaikan bootcamp kotakode x progate

NABILAH

Ada pepatah berbunyi "Tak kenal, maka tak sayang", perkenalkan nama saya adalah Nabilah dari Surabaya. Tahun ini saya akan berusia 21 tahun. Saya sangat tertarik dengan dunia visual. Selagi menunggu kelulusan, saya mengikuti bootcamp Progate x Kotakode untuk meningkatkan hardskill saya dalam dunia progamming dan menambah kenalan baru.

KAMALIYA

Nama saya adalah Kamaliya, dia lahir di Madura, 31 Desember 2000. saya memiliki hobi berolahraga, terutama basket. Bagi saya olahraga sangat kesehatan penting bagi tubuh. Di sini saya mengikuti program progate X kotakode untuk menambah ilmu di bidang programming. untuk pertama kalinya saya mencoba untuk keluar dari zona nyaman. Banyak hal yang saya dapat yaitu bagaimana bekerja sama dalam tim dan di sini saya bisa mempunyai banyak teman dari banyak daerah

BAYU

Jalan-jalan membeli baju, lebih murah kalau dipasar. Salam kenal saya Bayu, asal saya dari Denpasar. Saat ini saya bekerja sebagai programmer, disalah satu perusahaan swasta yang bergerak dibidang Teknologi Informasi dan Data Center yang berkantor pusat di Jakarta. Selain bekerja sebagai seorang programmer saya memiliki ketertarikan dibidang desain grafis. Bicara hobi, saya suka membaca manga serta menonton konser musik.

LILI

Halo, namaku Lila Istifarah Kusumaningrum, sering disapa Lila. Mahasiswi program pendidikan Manajemen Bisnis Syariah di salah satu universitas Islam negeri di Jawa Tengah. Saya mengikuti bootcamp Progate x Kotakode kali pertama ini untuk mengisi waktu libur semester untuk menambah pengalaman dan wawasan saya dalam bidang programming, dan untuk menambah teman.

YAHENDA

Nama saya farhan yehanda, mahasiswa teknologi informasi universitas sumatera utara asal Tanjungpinang, masuk ke dunia it ini membuat saya lebih teliti dalam mengerjakan sesuatu, semoga saya bisa bermanfaat bagi bangsa dan negara terimakasih.
